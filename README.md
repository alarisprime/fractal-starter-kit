# Fractal Starter Kit

## Development

### Setting up

```bash
$ yarn install # or $ npm install
```

### Development

Starts development server and watch tasks.

```bash
$ yarn start # or $ npm start
```

### Building

# Builds the style guide into '/dist' folder.

```bash
$ yarn build # or npm run build
```
